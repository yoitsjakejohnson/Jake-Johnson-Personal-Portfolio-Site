About Me

Hello, my name is Jake Johnson,

I am a Graphic Designer and Web Developer based in Green Bay, Wisconsin. I have a strong passion for design and front-end development.

I first discovered Photoshop in high school and started learning the basics of how to use the software. I then took Photoshop and Illustrator classes during high school. These expanded my knowledge of Adobe software and made me curious to keep learning more. Towards the end of high school they started a new HTML/CSS class. I took that class and found out I really enjoyed the process of coding HTML and watching the website come together in the end with the help of CSS.

After high school I then decided to enroll in the Web Design and Development program at Northeast Wisconsin Technical College. This program helped to give me the fundamental principles in web development. Some technologies I learned were HTML, CSS, Javascript, PHP, ASP.NET, and more.

While taking the web development course at NWTC I then truly realized how much I enjoyed the design aspect when it come to web development. After I graduated from the web development program I then decided to enroll in the Design and Graphic Technology program at NWTC. I am currently finishing up the final classes for that course before I graduate. Before taking this course I was already very comfortable using various pieces of software include in the Adobe Creative Cloud Suite. However, this program has solidified my skills even more when it comes to software such as Photoshop, Illustrator, and InDesign.

When it comes to web development the design step is very crucial. Some designers might feel like stepping straight into the development phase when creating a website for a client. They can definitely still have a successful outcome but creating a professional design before any coding begins really helps to create a fantastic responsive website in the end. With my design skills and background I can come up with a professional design following some of the latest web design trends while also knowing how to actually make the design come alive into a fully functioning website.
